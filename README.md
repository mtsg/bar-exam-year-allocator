# Bar-Exam-Year-Allocator

AI allocates the year of bar exam to be solved.


## Local development
1. Setup environments; Node.js<=18.19.1, yarn
1. Clone this repository.
1. ```yarn install```
1. ```yarn start```
1. See `localhost:3000` to check the Year allocated.
